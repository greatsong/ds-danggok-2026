모델링 — 직선을 긋다

🎯 오늘 만드는 것
  • 서울 기온 예측기 초안 — 연도를 슬라이더로 고르면 그해 예상 기온을 알려 주는 작은 AI

0. 지난 차시까지

  • 4차시에서 데이터를 믿을 수 있게 골라 냈다.
  • 관측일이 300일보다 적은 해(전쟁 시기·반쪽 관측)를 빼니 1908~2025년 중 114개 연도가 남았다.
  • 이제 이 깨끗한 점들 위에 직선을 긋는다.

1. 회귀 = 점들에 가장 잘 맞는 직선 긋기

  • 가로축 연도, 세로축 연평균기온으로 점을 찍으면 우상향하는 구름이 보인다.
  • 그 구름 한가운데를 지나는 한 줄기 직선을 찾는 일이 회귀다.
  • 직선의 기울기가 곧 "100년에 몇 도 오르는가"를 뜻한다.
💡 기울기를 읽는 법
  • 이 데이터로 직선을 그으면 기울기가 +2.60℃/100년이 나온다.
  • 뜻: 지난 114년 동안 연평균기온이 100년당 평균 2.60℃씩 오르는 경향.
  • 한 해 한 해의 오르내림이 아니라 긴 흐름의 방향을 요약한 숫자다.

2. "학습"이란 — model.fit 한 줄

  • 가장 잘 맞는 직선을 사람이 손으로 찾지 않는다. 컴퓨터가 데이터를 보고 스스로 찾는다.
  • 이 "스스로 규칙을 찾는 과정"이 학습이고, 코드로는 model.fit() 딱 한 줄이다.
  • 학습이 끝난 직선에 새 연도를 넣으면 예측이 나온다 → model.predict().
단계하는 일
학습점들에 가장 잘 맞는 직선 찾기 (model.fit)
예측학습한 직선에 연도를 넣어 기온 뽑기 (model.predict)

  • 데이터 = seoul.csv 주소 그대로 불러오기(다운로드 없음, 매일 아침 자동 갱신).

① 샘플 프롬프트
서울 기온으로 미래를 예측하는 스트림릿 앱(main.py) 하나를 만들어 줘. 스트림릿 클라우드에 올릴 거야. 조건:

- 데이터는 이 주소에서 불러와(1907년부터 어제까지의 일별 기온이고,
  열은 날짜, 지점, 평균기온, 최저기온, 최고기온이야):
  https://raw.githubusercontent.com/greatsong/modudata/main/data/seoul.csv
- 날짜에서 연도를 뽑아 연평균기온으로 묶어 줘.
- 아직 진행 중인 올해와, 관측일이 300일보다 적은 해는 평균을 믿기
  어려우니 빼고 학습해 줘.
- 연도로 연평균기온을 예측하는 회귀 직선을 학습시켜 줘.
- 슬라이더로 1900~2100년 사이 연도를 고르면 예상 기온을 크게 보여 줘.
- 실제 점들과 학습한 직선을 plotly 그래프로 함께 그려 줘.
- 필요한 라이브러리 목록(requirements.txt)도 같이 줘. 버전 숫자 없이 이름만,
  스트림릿·판다스·넘파이는 기본 설치라 빼고.
- 한국어 라벨, 따뜻한 톤, 초보자용 한국어 주석. main.py 전체 코드를 한 번에 줘.
  • AI가 아래 모습의 코드를 준다(똑같지 않아도 된다).
② AI가 준 코드 예시
import streamlit as st
import pandas as pd
from sklearn.linear_model import LinearRegression

st.title("🌡️ 서울 기온 예측기")

# 수집 — CSV 불러오기 (1907년~어제, 일별 기온 · 매일 자동 갱신)
url = "https://raw.githubusercontent.com/greatsong/modudata/main/data/seoul.csv"
df = pd.read_csv(url)

# 전처리 — 연도별 평균으로 묶고, 진행 중인 올해와
#          관측일 300일 미만인 해는 빼기
df["연도"] = pd.to_datetime(df["날짜"]).dt.year
yearly = df.groupby("연도")["평균기온"].agg(["mean", "count"]).reset_index()
yearly = yearly[(yearly["연도"] < df["연도"].max()) & (yearly["count"] >= 300)]
yearly = yearly.rename(columns={"mean": "평균기온"})

X = yearly[["연도"]]   # 대괄호 두 개: 표 모양으로
y = yearly["평균기온"]

# 학습 — 이 한 줄이 '학습'! 가장 잘 맞는 직선을 찾는다
model = LinearRegression()
model.fit(X, y)

# 예측 — 슬라이더로 연도를 넣으면 그해 기온을 예측
year = st.slider("예측할 연도", 1900, 2100, 2050)
pred = model.predict([[year]])[0]
st.subheader(f"{year}년 예측 평균기온: {pred:.1f}℃")

# (지면상 생략 — 실제 코드엔 plotly로 그린 '실제 점 + 학습 직선' 그래프가 함께 옵니다)
  • 핵심은 model.fit(X, y) 한 줄 — 여기서 작은 AI가 태어난다.
  • 슬라이더를 움직이면 예측값이 따라 바뀐다. 여러 연도를 넣어 보자.
③ 프롬프트 개선 예시
좋아, 이어서 고쳐 줘.
- 직선의 기울기를 '100년에 몇 도 오르는가'로 바꿔서 화면에 크게 보여 줘.
- 예측 기온 옆에, 가장 최근 해보다 몇 도 오르는지도 함께 보여 줘.
- 실제 점은 회색, 학습한 직선은 주황색으로.
⚠️ 슬라이더를 2100년까지 밀면?
  • 직선이라 밀수록 예측값이 끝없이 올라간다.
  • 데이터에 없는 먼 미래까지 직선을 늘이는 것을 외삽이라 한다.
  • 정말 그렇게 오를까? 이 의심이 다음 차시(평가와 한계)의 중심이다.
