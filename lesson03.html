시각화 · 그래프에서 구멍 발견

🎯 오늘 만드는 것
  • 연평균 추세 그래프 — 1907년부터 지금까지 서울 기온의 흐름
  • 이상 지점 메모 — 그래프에서 이상해 보이는 곳 2군데 기록

0. 지난 차시

  • 표의 첫 인상 · 관측 개수가 날짜 수보다 적다는 낌새(결측 약 756개)
  • 오늘 → 숫자를 그림으로 바꿔 흐름을 본다

1. 일별을 연평균으로

  • 매일 값은 4만 줄 · 그대로는 흐름이 안 보임
  • 같은 해끼리 묶어 연평균으로 요약 → 100년 넘는 추세가 한 줄 그래프로

2. 연평균 추세 그래프 (①②③)

① 샘플 프롬프트
서울 일별 기온을 연평균으로 바꿔서 1907년부터 지금까지의 추세를 선 그래프로 그려 줘.
가로축은 연도, 세로축은 연평균기온으로.
데이터는 여기야: https://raw.githubusercontent.com/greatsong/modudata/main/data/seoul.csv
  • 여러분 화면이 이런 모습이면 성공 · 글자까지 같을 필요 없음
② AI가 준 코드 예시
import streamlit as st
import pandas as pd
import plotly.express as px

st.title("📈 서울 연평균기온 추세")

# 일별 기온 불러오기
url = "https://raw.githubusercontent.com/greatsong/modudata/main/data/seoul.csv"
df = pd.read_csv(url)

# 날짜에서 연도를 뽑아 같은 해끼리 평균 내기
df["연도"] = pd.to_datetime(df["날짜"]).dt.year
yearly = df.groupby("연도")["평균기온"].mean().reset_index()

# 연도(가로) · 연평균기온(세로) 선 그래프
fig = px.line(yearly, x="연도", y="평균기온", title="서울 연평균기온 (1907~현재)")
st.plotly_chart(fig, use_container_width=True)
  • 그래프가 뜨면 → 세 박자대로 ③ 개선으로 이상한 곳을 드러내기
③ 프롬프트 개선 예시
좋아! 그래프에서 비어 있는 연도와 유난히 낮은 연도가 눈에 띄게
색이나 표시를 넣어 줘. 어디가 이상한지 한눈에 보이게.

3. 구멍 찾기 — 모둠 관찰

🔎 오늘의 발견 (★) — 그래프에 구멍이 있다
  • 전체 흐름은 우상향(더워지는 쪽) · 그런데 두 곳이 이상하다
  • ① 1950년대 초 = 선이 끊긴다 · 1951·1952년은 연평균 값이 아예 없다
  • ② 1953년 = 한 점이 뚝 떨어진다 · 다른 해는 11~12℃인데 0.6℃
  • "데이터가 왜 이렇게 생겼지?" → 이 물음이 다음 차시의 정체 밝히기로 이어진다
✔️ 모둠 토의 · 이상 지점 2곳 기록
  • 그래프에서 이상해 보이는 곳 2군데를 찾아 연도와 함께 적는다
  • 각각 왜 그럴지 추측 한 줄씩 (정답 아니어도 됨 · 추측이 재료)
  • 이 메모가 다음 차시 전처리의 출발점
💡 구멍이 안 보이면
  • 1940~1960년 구간만 확대해 보면 끊긴 선과 떨어진 점이 또렷하다
  • 정상과 이상을 나누는 눈 = "다른 값과 유난히 다른가"를 보는 것
