- 데이터를 학습용과 시험용으로 나눈 뒤, 시험용에서 정확도를 재서

② AI가 준 코드 (예시)
import streamlit as st
import pandas as pd
from sklearn.ensemble import RandomForestClassifier
from sklearn.model_selection import train_test_split
from sklearn.metrics import accuracy_score

st.title("🚇 강남역, 내일 붐빌까?")

# 데이터 불러오기 — 서울 23곳, 매시간 혼잡도
url = "https://raw.githubusercontent.com/greatsong/modudata/main/data/seoul_congestion_log.csv"
df = pd.read_csv(url)

# 시각에서 요일과 시간을 뽑기
df["시각"] = pd.to_datetime(df["시각"])
df["요일"] = df["시각"].dt.dayofweek   # 월=0 … 일=6
df["시간"] = df["시각"].dt.hour

# 입력(특징): 장소·요일·시간 / 정답(레이블): 혼잡도
X = pd.get_dummies(df[["지역명", "요일", "시간"]], columns=["지역명"])
y = df["혼잡도"]

# 학습용·시험용으로 나누기 (4분의 1은 시험용)
X_train, X_test, y_train, y_test = train_test_split(
    X, y, test_size=0.25, random_state=42)

# 랜덤 포레스트로 학습 — 이 한 줄이 '학습'
model = RandomForestClassifier(random_state=42)
model.fit(X_train, y_train)

# 시험용으로 정확도 재기
acc = accuracy_score(y_test, model.predict(X_test))
st.metric("정확도", f"{acc:.1%}")
💡 방금 무슨 일이?
  • 정답이 있는 데이터로 배웠다 = 지도학습 (회귀도 지도학습이었음)
  • 회귀의 model.fit()과 똑같은 한 줄로 학습 — 바뀐 건 "숫자" 대신 "단계"를 맞힌다는 것뿐
  • 첫 정확도는 약 88.8% — 열에 아홉은 맞힌 셈
  • 단계별로 얼마나 맞혔는지 더 보고 싶으면 대화로 개선 ↓
