1. seoul.csv — 어떤 표인가

📦 오늘 쓸 데이터
  • 파일 = seoul.csv · 1907년부터 어제까지의 서울 일별 기온 · 매일 아침 자동 갱신
  • 열 = 날짜 · 지점 · 평균기온 · 최저기온 · 최고기온
  • 주소 = https://raw.githubusercontent.com/greatsong/modudata/main/data/seoul.csv · 다운로드 없이 주소로 바로 불러옴
  • 큰 표는 통째로 보지 않는다 · 두 가지만 눈에 익히기
보는 법무엇을 보나
앞부분 몇 줄열 이름·값의 생김새 (표가 어떻게 생겼나)
요약통계개수·평균·최소·최대 (표의 첫 인상)

2. 표의 첫 인상 뽑기 (①②③)

① 샘플 프롬프트
서울 기온 데이터(seoul.csv)를 스트림릿으로 불러와서 표의 앞부분과 요약통계를 보여 줘.
관측 개수, 평균, 최고, 최저기온이 한눈에 보이게 해 줘.
주소는 여기야: https://raw.githubusercontent.com/greatsong/modudata/main/data/seoul.csv
  • 여러분 화면이 이런 모습이면 성공 · 글자까지 같을 필요 없음
② AI가 준 코드 예시
import streamlit as st
import pandas as pd

st.title("🌡️ 서울 기온 데이터 첫 인상")

# 표 불러오기 (1907년~어제, 일별 기온)
url = "https://raw.githubusercontent.com/greatsong/modudata/main/data/seoul.csv"
df = pd.read_csv(url)

# 앞부분 보기 — 표가 어떻게 생겼나
st.subheader("표의 앞부분")
st.dataframe(df.head())

# 요약통계 — 개수·평균·최소·최대 한눈에
st.subheader("숫자 요약")
st.dataframe(df.describe())
  • 결과를 보고 → 세 박자대로 ③ 개선으로 한 걸음 더
③ 프롬프트 개선 예시
좋아! 여기에 하나 더 더해 줘.
열마다 빠진 값(관측이 없는 날)이 몇 개인지도 함께 표로 보여 줘.
✔️ 요약통계 읽는 법
  • 개수(count) = 값이 몇 개 있나 (빈 칸은 안 셈)
  • 평균(mean) = 다 더해 개수로 나눈 값 · 최소·최대 = 가장 낮은/높은 값
  • 서울 평균기온 = 약 11.8℃ · 최저 -19.2℃ · 최고 33.7℃ · 폭이 50도가 넘음
🔎 오늘의 발견 — 관측 개수가 날짜 수보다 적다
  • 전체 행은 약 42,958줄인데, 평균기온 값은 42,202개뿐
  • 차이 756 = 빠진 값(결측) · 어떤 날은 기온이 기록되지 않았다
  • "왜 어떤 날은 비어 있지?" → 이 물음이 다음 두 차시의 구멍 이야기로 이어진다
